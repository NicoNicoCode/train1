配置CentOS的网络环境和SSH登陆
=============================
<br/>

1.修改静态IP
------------
　　使用“ip addr show”显示出本机的静态IP，可以看出，本机的网络名称为ens33
    <br/>
　　![](1.png)
    <br/>

2.修改网络配置
--------------
　　使用“vi /etc/sysconfig/network-scripts/ifcfg-ens33”,进入网络配置文件，将ONROOT修改为true.
    <br/>
　　![](2.png)
    <br/>

3.SSH登陆
---------
　　下载Xshell 5,在新建连接的属性输入虚拟机的IP，端口号与ens33里的保持一致。
    <br/>
　　![](3.png)
    <br/>

4.验证
---------
　　在用户身份验证输入虚拟机的用户名和密码。并启动连接，显示成功登陆
    <br/>
　　![](4.png)
    <br/>
　　![](5.png)
